1. Создать папку для исходников: `src`
1. Создать файл исходного кода: `main.c`
1. Создать файл проекта: `СMakeLists.txt`
1. Конфигурирование сборки (из папки `src`):
   ```bash
   cmake -B ../build . 
   ```
   будет создана папка `build`
1. Сборка
   ```bash
   cmake --build ../build 
   ```
1. Запуск программы
   ```bash
   ../build/hello_cmake
   ```
