  ```g++ --version```
3. Создайте в Sublime Text файл с расширением cpp и исходным кодом простой    программы на C++:
  ```C++

  int main(){
	   cout << "Hello, World!" << endl;
	    return 0;
    }
```

```g++ имя-файла-с-иходным-кодом -o имя-exe-файла```\
